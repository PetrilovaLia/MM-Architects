# MM Architects Portfolio

A responsive portfolio website for MM Architects, showcasing architectural projects with a modern, minimalist design. The website features a clean layout optimized for both desktop and mobile devices.

## Features

- **Responsive Design**: Fully responsive layout that adapts to all screen sizes
- **Project Gallery**: Showcase of architectural projects with images
- **Mobile Optimization**: Custom CSS grid layout for mobile compatibility
- **Clean UI**: Minimalist design focused on visual content
- **Contact Form**: Easy way for potential clients to get in touch
- **Fast Loading**: Optimized assets for quick page load times

## Tech Stack

**Frontend**
- HTML5
- CSS3 (with Grid and Flexbox)
- JavaScript

**Design**
- Responsive web design principles
- Mobile-first approach
- Modern CSS techniques

## Installation

1. Clone the repository
```bash
git clone https://github.com/PetrilovaLia/mm-architects.git
cd mm-architects
```

2. Open the project
Simply open `index.html` in your web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

3. View the website
```
http://localhost:8000
```
